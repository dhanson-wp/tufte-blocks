# Tufte Blocks

A WordPress block theme inspired by Edward Tufte's design principles. Typography-first, with generous margins, ET Book serif type, and minimal ornamentation. Features dark mode by default with a light variation. Built for writers, researchers, and academics who believe the content should always come first.

## Features

- **Typography-first design** — ET Book serif, optimal line length, generous margins
- **Dark mode default** — Easy-on-the-eyes dark palette with gold accents
- **Light mode style variation** — Switch to light via Appearance → Editor → Styles
- **Block patterns** — Header, footer, about card, reading list, newsletter CTA, syllabus, and more
- **Full-site editing** — Customize templates and styles in the Site Editor
- **Minimal ornamentation** — Content-first, Tufte-inspired aesthetic

## Requirements

- WordPress 6.4 or higher
- PHP 7.4 or higher

## Installation

1. Download the [latest release](https://github.com/dhanson-wp/tufte-blocks/releases) or clone this repository into `wp-content/themes/`
2. Activate the theme in **Appearance → Themes**
3. Optional: switch to the light style variation in **Appearance → Editor → Styles**
